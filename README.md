# political compass

8-axis political compass test. 50 questions, ~10 min.

## axes

- economic (equality vs markets)
- civil (liberty vs authority)
- diplomatic (nation vs globe)
- social (tradition vs progress)
- technology (caution vs acceleration)
- environment (growth vs ecology)
- governance (local vs central)
- justice (rehabilitate vs punish)

## stack

- sveltekit
- tailwindcss v4
- threlte (three.js)

## run

```sh
pnpm i
pnpm dev
```

## features

- multi-weight scoring system (each question affects 2-4 axes)
- 2d compass, radar chart, 3d visualization
- compare results to 20 historical figures
- shareable results via url
- auto-saves progress locally
